## 🚗 Mobility Hub - 스마트 주차장 모빌리티 허브

RC카 라인트레이싱, MQTT 통신, 사용자/관리자 웹을 결합한  
**스마트 주차장 통합 제어 시스템**

### 프로젝트 개요

웹에서 차량 등록 및 서비스 요청을 수행하고,  
RC카 경로 전달과 구역 상태 갱신, 게이트/카메라/펌프/리프트 제어까지  
하나의 흐름으로 연결한 모빌리티 허브 프로젝트입니다.

### 주요 기능

- 사용자/관리자 로그인 및 권한 분기
- 차량 등록 및 서비스 요청 처리
- RC카 경로 전달 및 위치 보고
- MQTT 기반 게이트 / 카메라 / 펌프 / 리프트 제어
- 관리자 웹에서 상태 모니터링 및 제어

### 담당 업무

- 출입구(게이트/카메라) 기능 구현
- 입출구 Backend 및 Frontend 구현
- API 연동 및 데이터 전처리
- Admin Web 보완
- 입출구 조형물 제작 참여

## 🏢 Building RCS - 빌딩 원격관리 시스템

스마트 빌딩 환경에서 디바이스 원격 제어와 자동 데이터 수집을 구현한  
**Java 기반 IoT 원격관리 시스템**

### 프로젝트 개요

관리자와 사용자 권한을 구분하여  
출입문, 엘리베이터, 주차장, 조명, 화재 감지 기능 등을  
MQTT와 DB 연동으로 제어 및 모니터링할 수 있도록 구현한 프로젝트입니다.

### 주요 기능

- 로그인 / 회원가입 기능
- 출입문 원격 제어
- 엘리베이터 호출 및 이동 제어
- 사무실별 디바이스 제어
- 주차장 입출차 관리 및 로그 조회
- 온습도 / 가스 기반 화재 감지 및 경보 기능

### 담당 업무

- 주차 관리 기능 구현
- 센서 로그 수집 기능 구현
